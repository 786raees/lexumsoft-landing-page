# LinkedIn Post Writer Memory

## Character Count Tips
- `wc -c` on the extracted code block content gives a reliable count (includes newlines as chars)
- First drafts consistently run 400-500 chars over target. Start writing with the mindset of ~2,500 chars of actual prose to land at 2,800-2,950 after separators/hashtags
- Easiest places to trim without losing impact: problem elaboration (middle), empathy section filler, solution bridge wordiness
- Never trim from: hook (lines 1-3), client anecdote specifics, CTA section
- Hashtags with 10 tags = ~160-170 chars; 11 tags = ~190-200 chars; each `---` separator with surrounding newlines = ~6-8 chars total

## Post Arc Timing
- The math section is a crowd favorite -- make the dollar gap dramatic but realistic (grounded in agency bible ranges)
- Empathy section works best at 2-3 sentences max. Longer loses momentum before the anecdote.
