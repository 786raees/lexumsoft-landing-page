## Notes

- Upwork allows up to 15 skill tags on the main profile. All 15 above should be added.
- Specialized profiles allow 5 tags each. Use the prioritization above when setting those up.
- Tags are used by Upwork's search algorithm to match freelancers to jobs. The first few tags carry more weight.
- "WordPress" and "Web Design" rank highest because most contractor clients searching Upwork use these terms.
- "Google My Business" is Upwork's tag name for Google Business Profile work.
